# About

A viewer for 3D visualization of NYU-collected data on egress behavior from NYC COVID-19 exposed health facilities.

# Preparing

 * Add `data/` folder tracked outside of repo due to size
 * `npm install`

# Running

 * `npm start`
 * Go to `http://localhost:1234/src
